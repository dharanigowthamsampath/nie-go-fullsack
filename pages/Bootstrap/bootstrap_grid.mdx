## âœ… **2. Bootstrap Grid Classes**

Bootstrap provides **five responsive breakpoints**:  
| Breakpoint | Class Prefix | Screen Size |
|------------|-------------|-------------|
| Extra Small | `.col-` | Less than 576px (default for mobile) |
| Small | `.col-sm-` | 576px and up |
| Medium | `.col-md-` | 768px and up |
| Large | `.col-lg-` | 992px and up |
| Extra Large | `.col-xl-` | 1200px and up |
