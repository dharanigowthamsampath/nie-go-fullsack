# 📌 Exploring Bootstrap Docs

Bootstrap is a popular front-end framework used for building responsive and mobile-first web applications. It provides pre-styled components, a grid system, and utility classes to speed up development.

## 📌 Latest Bootstrap Version

The latest version of Bootstrap is **Bootstrap 5.3**.

### 🔗 Official Links:

- 📖 [Bootstrap Documentation](https://getbootstrap.com/docs/5.3/getting-started/introduction/)
- 📦 [Bootstrap CDN](https://www.bootstrapcdn.com/)
- 🏗 [Bootstrap GitHub Repository](https://github.com/twbs/bootstrap)
- 🎨 [Bootstrap Icons](https://icons.getbootstrap.com/)

## 📌 Key Features of Bootstrap 5.3

- Responsive Grid System
- Prebuilt UI Components (Buttons, Modals, Forms, etc.)
- Utility Classes for Quick Styling
- JavaScript Plugins (Dropdowns, Carousels, Tooltips)
- Improved CSS with Flexbox & Grid
