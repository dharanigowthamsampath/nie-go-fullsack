## ðŸŽ¨ Basic CSS Syntax


### Explanation:

- **Selector**: Targets an HTML element (e.g., `h1`, `p`, `.class`, `#id`).
- **Property**: Specifies the style attribute (e.g., `color`, `font-size`).

## ðŸ“Œ Applying CSS

### Inline CSS

```html copy filename="index.html"
<p style="color: blue; font-size: 18px;">This is a blue paragraph.</p>
```

### Internal CSS

```html copy filename="index.html"
<style>
  p {
    color: blue;
    font-size: 18px;
  }
</style>
```

### External CSS

```html copy filename="index.html"
<link rel="stylesheet" href="styles.css" />
```

### Explanation:

- **Inline CSS**: Applied directly inside an HTML tag.
- **Internal CSS**: Defined inside a `<style>` block within the `<head>`.
- **External CSS**: A separate `.css` file linked to the HTML page.

## ðŸ“Œ Selectors

### Universal Selector

```html copy filename="index.css"

```

### Element Selector

```html copy filename="index.css"

```

### Class Selector

```html copy filename="index.css"

```

### ID Selector

```html copy filename="index.css"

```

### Explanation:

- `*`: Selects all elements.
- `element`: Targets specific HTML elements.
- `.class`: Targets elements with a specific class.
- `#id`: Targets elements with a specific ID.
