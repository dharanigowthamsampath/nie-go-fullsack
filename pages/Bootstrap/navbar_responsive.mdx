# ğŸ“Œ Building a Responsive Navbar with Bootstrap

A **responsive navbar** adjusts based on screen size, ensuring a great user experience on all devices. Bootstrap provides built-in classes to make this process simple.

## ğŸ”¹ Step 1: Include Bootstrap

First, include Bootstrap in your project via **CDN**:

```html copy
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <title>Responsive Navbar</title>
    <!-- Bootstrap CSS -->
    <link
      href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css"
      rel="stylesheet"
    />
    <!-- Bootstrap JS (for toggling navbar) -->
    <script
      src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"
      defer
    ></script>
  </head>
  <body></body>
</html>
```

### âœ… Explanation:

- The **Bootstrap CSS file** styles the navbar.
- The **Bootstrap JS file** enables interactivity (e.g., toggling the menu on small screens).
- The **`meta viewport` tag** ensures responsiveness.

## ğŸ”¹ Step 2: Create the Navbar Structure

Now, let's create a **basic responsive navbar**:

```html copy
<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
  <div class="container-fluid">
    <!-- Brand Logo -->
    <a class="navbar-brand" href="#">MyBrand</a>

    <!-- Toggler Button (for mobile view) -->
    <button
      class="navbar-toggler"
      type="button"
      data-bs-toggle="collapse"
      data-bs-target="#navbarContent"
    >
      <span class="navbar-toggler-icon"></span>
    </button>

    <!-- Navbar Links -->
    <div class="collapse navbar-collapse" id="navbarContent">
      <ul class="navbar-nav ms-auto">
        <li class="nav-item"><a class="nav-link" href="#">Home</a></li>
        <li class="nav-item"><a class="nav-link" href="#">About</a></li>
        <li class="nav-item"><a class="nav-link" href="#">Services</a></li>
        <li class="nav-item"><a class="nav-link" href="#">Contact</a></li>
      </ul>
    </div>
  </div>
</nav>
```

### âœ… Explanation:

- `<nav class="navbar navbar-expand-lg navbar-dark bg-dark">`:

  - **`navbar`** â†’ Creates a Bootstrap navbar.
  - **`navbar-expand-lg`** â†’ Navbar is fully expanded on large screens (`lg` breakpoint).
  - **`navbar-dark bg-dark`** â†’ Dark theme with dark background.

- `<a class="navbar-brand" href="#">MyBrand</a>`:

  - **Navbar brand/logo** linking to the homepage.

- `<button class="navbar-toggler">`

  - **Toggler button** appears on small screens.
  - **`data-bs-toggle="collapse"`** â†’ Enables the toggle effect.
  - **`data-bs-target="#navbarContent"`** â†’ Connects to the collapsible section.

- `<div class="collapse navbar-collapse" id="navbarContent">`

  - **Collapsible container** for navbar links.
  - **Hides links on small screens and expands on large screens**.

- `<ul class="navbar-nav ms-auto">`
  - **`navbar-nav`** â†’ Styles the list as a navigation menu.
  - **`ms-auto`** â†’ Aligns links to the right.
