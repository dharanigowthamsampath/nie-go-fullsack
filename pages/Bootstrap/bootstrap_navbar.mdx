# ðŸ“Œ Building a Navbar Using Bootstrap

Bootstrap provides a simple and effective way to create responsive navigation bars. In this guide, we'll go step by step to build a basic navbar using Bootstrap.

## ðŸ”¹ Step 1: Include Bootstrap in Your Project

Before using Bootstrap components, you need to include Bootstrap in your project. You can do this by adding the Bootstrap CDN link in the `<head>` section of your HTML file:

```html copy
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Bootstrap Navbar</title>
    <link
      href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css"
      rel="stylesheet"
    />
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
  </head>
  <body></body>
</html>
```

## ðŸ”¹ Step 2: Create a Basic Navbar

Now, let's create a basic navigation bar using the `navbar` class:

```html copy
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">MyWebsite</a>
    <button
      class="navbar-toggler"
      type="button"
      data-bs-toggle="collapse"
      data-bs-target="#navbarNav"
    >
      <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarNav">
      <ul class="navbar-nav">
        <li class="nav-item">
          <a class="nav-link active" href="#">Home</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">About</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Services</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Contact</a>
        </li>
      </ul>
    </div>
  </div>
</nav>
```

### Explanation:

- `navbar navbar-expand-lg navbar-light bg-light`: Creates a responsive, light-themed navigation bar.
- `container-fluid`: Ensures the navbar stretches across the full width.
- `navbar-brand`: Defines the website or brand name.
- `navbar-toggler`: Adds a collapsible menu button for mobile screens.
- `navbar-collapse`: Wraps the menu items to collapse them on smaller screens.
- `navbar-nav`: A flex container for navigation links.
- `nav-item` and `nav-link`: Define individual navigation items and links.

## ðŸ”¹ Step 3: Adding a Dropdown Menu

To add a dropdown menu inside the navbar:

```html copy
<li class="nav-item dropdown">
  <a
    class="nav-link dropdown-toggle"
    href="#"
    id="navbarDropdown"
    role="button"
    data-bs-toggle="dropdown"
  >
    More
  </a>
  <ul class="dropdown-menu">
    <li><a class="dropdown-item" href="#">Gallery</a></li>
    <li><a class="dropdown-item" href="#">Blog</a></li>
  </ul>
</li>
```

### Explanation:

- `dropdown`: Creates a dropdown container.
- `dropdown-toggle`: Adds a toggle button for the dropdown.
- `data-bs-toggle="dropdown"`: Enables Bootstrap's dropdown functionality.
- `dropdown-menu`: Contains the list of dropdown items.
- `dropdown-item`: Defines each clickable item in the dropdown.

## ðŸ”¹ Step 4: Adding a Search Bar

To add a search bar to the navbar:

```html copy
<form class="d-flex">
  <input class="form-control me-2" type="search" placeholder="Search" />
  <button class="btn btn-outline-success" type="submit">Search</button>
</form>
```

### Explanation:

- `d-flex`: Makes the form elements display flexibly in a row.
- `form-control`: Styles the input field.
- `btn btn-outline-success`: Styles the search button.

## ðŸ”¹ Step 5: Making the Navbar Sticky

To make the navbar stick to the top while scrolling, add `fixed-top`:

```html copy
<nav class="navbar navbar-expand-lg navbar-light bg-light fixed-top"></nav>
```

### Explanation:

- `fixed-top`: Makes the navbar stay at the top of the page when scrolling.
