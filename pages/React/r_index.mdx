# ⚛️ Overview of React

## What is React?

React is a **JavaScript library** for building user interfaces. Developed by Facebook, it allows developers to create reusable UI components and manage state efficiently.

## 🌟 Why Use React?

✅ **Component-Based Architecture** - Reusable UI components.
✅ **Virtual DOM** - Improves performance by updating only changed parts.
✅ **Fast and Efficient** - Handles UI updates smoothly.
✅ **One-Way Data Flow** - Predictable state management.
✅ **Huge Community Support** - Many open-source libraries and tools.

## 🏗️ Core Features

- **JSX**: A syntax extension for writing UI in JavaScript.
- **Components**: Modular, reusable UI elements.
- **State & Props**: Manage and pass data efficiently.
- **Hooks**: Enable functional components to manage state and lifecycle.
- **React Router**: Handle navigation in single-page applications.
- **Context API & Redux**: State management solutions for large applications.

## 📦 How to Get Started?

1. **Install React using Create React App:**
   ```sh
   npx create-react-app my-app
   cd my-app
   npm start
   ```
2. **Install React via Vite (Faster Setup):**
   ```sh
   npm create vite@latest my-app --template react
   cd my-app
   npm install
   npm run dev
   ```
