## **ðŸ“Œ Step 3: Create a Database Connection**

Inside the **`main.go`** file, add the **`connectDB()`** function:

```go filename="main.go" copy
func connectDB() error {
	// Set MongoDB client options
	clientOptions := options.Client().ApplyURI(mongoURI)

	// Create a MongoDB client
	var err error
	client, err = mongo.Connect(context.Background(), clientOptions)
	if err != nil {
		return err
	}

	// Check if connection is successful
	err = client.Ping(context.Background(), nil)
	if err != nil {
		return err
	}
	return nil
}
```

### **ðŸ”¹ Explanation:**

âœ… Connects to MongoDB at **`mongodb://localhost:27017`**  
âœ… Uses **context.Background()** to manage the database connection  
âœ… Checks if MongoDB is reachable using `Ping()`

## **ðŸ“Œ Step 4: Define a Collection and Schema**

MongoDB stores data as **JSON-like documents** (BSON format). We will create a `users` collection inside a `go_db` database.

```go filename="main.go" copy
// Define a struct for the user document
type User struct {
	ID   string `bson:"_id,omitempty"`
	Name string `bson:"name"`
	Age  int    `bson:"age"`
}

// Get the collection reference
func getCollection() *mongo.Collection {
	return client.Database("go_db").Collection("users")
}
```

## **ðŸ“Œ Step 6: Retrieve (Read) Users**

```go filename="main.go" copy
func getAllUsers() {
	collection := getCollection()

	// Retrieve all users
	cursor, err := collection.Find(context.Background(), bson.M{})
	if err != nil {
		log.Fatal("Error fetching users:", err)
	}
	defer cursor.Close(context.Background())

	var users []User
	for cursor.Next(context.Background()) {
		var user User
		cursor.Decode(&user)
		users = append(users, user)
	}

	fmt.Println("ðŸ‘¥ Users in Database:")
	for _, user := range users {
		fmt.Printf("ID: %s, Name: %s, Age: %d\n", user.ID, user.Name, user.Age)
	}
}
```

### **ðŸ”¹ Explanation:**

âœ… Uses **Find()** to retrieve all users  
âœ… Decodes MongoDB BSON into a **Go struct**  
âœ… Prints all users stored in the database

```go filename="main.go" copy
func updateUser(name string, newAge int) {
	collection := getCollection()
```
