## **Step 4: Create Your First Gin Server**

Create a new file **`main.go`** inside the `gin-app` folder and add the following code:

### **Code: Simple Gin Server**

```go filename="main.go" copy
package main

import (
    "github.com/gin-gonic/gin"
)

func main() {
    // Create a new Gin router
    router := gin.Default()

    // Define a GET route
    router.GET("/", func(c *gin.Context) {
        c.JSON(200, gin.H{"message": "Hello, World!"})
    })

    // Start the server on port 8080
    router.Run(":8080")
}
```

### **Explanation**

1. **Import Gin** â†’ The `gin-gonic/gin` package is imported.
2. **Create a Router** â†’ `gin.Default()` initializes a Gin router with logging and recovery middleware.
3. **Define a Route** â†’ `router.GET("/", handler)` sets up a simple API endpoint that returns `Hello, World!` as JSON.
4. **Run the Server** â†’ `router.Run(":8080")` starts the server on **localhost:8080**.

## **Step 6: Adding More Routes**

Letâ€™s add more routes to handle different requests.

### **Code: Handling Multiple Routes**

```go filename="main.go" copy
package main

import (
    "github.com/gin-gonic/gin"
)

func main() {
    router := gin.Default()

    // Home route
    router.GET("/", func(c *gin.Context) {
        c.JSON(200, gin.H{"message": "Welcome to Gin!"})
    })

    // Dynamic Route
    router.GET("/user/:name", func(c *gin.Context) {
        name := c.Param("name")
        c.JSON(200, gin.H{"user": name})
    })

    // Query Parameters
    router.GET("/search", func(c *gin.Context) {
        query := c.Query("q")
        c.JSON(200, gin.H{"query": query})
    })

    router.Run(":8080")
}
```

### **Explanation**

1. **`/user/:name`** â†’ Accepts a dynamic parameter (e.g., `/user/dharani`).
2. **`/search?q=gin`** â†’ Handles query parameters (e.g., `/search?q=gin`).

Test these by visiting:

- [http://localhost:8080/user/dharani](http://localhost:8080/user/dharani)
- [http://localhost:8080/search?q=gin](http://localhost:8080/search?q=gin)
