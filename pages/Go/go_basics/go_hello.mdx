# **Simple Hello World Program in Go**

## **Introduction**

Go (also known as Golang) is a statically typed, compiled programming language designed for simplicity and efficiency. It is widely used for system programming, cloud applications, and web development due to its speed and concurrency support.

In this blog, we will write a simple **"Hello, World!"** program in Go and explain its components step by step.

---

## **Writing Your First Go Program**

To begin, create a new file called `main.go` and add the following code:

### **Code: Hello World in Go**

```go filename="main.go" copy
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

## **Explanation of the Code**

### **1. `package main`**

```go filename="main.go" copy
package main
```

- In Go, every program is organized into packages.
- The `main` package is the entry point of every standalone Go application.
- Without this, the Go compiler wonâ€™t know where to start execution.

### **2. `import "fmt"`**

```go filename="main.go" copy
import "fmt"
```

- `import` is used to include built-in or third-party packages in a Go program.
- The `"fmt"` package stands for **format** and provides functions for formatted I/O (input and output).
- Here, we use it to print text to the console.

### **3. `func main()`**

```go filename="main.go" copy
func main() {
```

- The `main` function is the starting point of execution in a Go program.
- Every Go program must have a `main()` function inside the `main` package.
- Without this function, the program wonâ€™t run.

### **4. Printing to the Console**

```go filename="main.go" copy
    fmt.Println("Hello, World!")
```

- `fmt.Println()` is used to print text to the console.
- `"Hello, World!"` is the message that will be displayed when the program runs.
- `Println` adds a new line (`\n`) at the end of the output automatically.
