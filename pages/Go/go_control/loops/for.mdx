# **`for` Loop in Go**

## **Introduction**

The `for` loop in Go is used to execute a block of code **multiple times** based on a condition.

- Go **only has `for`** (no `while` or `do-while`).
- It can be used in **three different ways**:
  - **Basic `for` loop (like `for` in C, Java, Python)**
  - **`for` as a `while` loop**
  - **Infinite `for` loop**

---

## **Syntax of `for` Loop**

```go filename="main.go" copy
for initialization; condition; increment/decrement {
    // Code inside the loop
}
```

### **Explanation**

1. **Initialization**: Runs once before the loop starts.
2. **Condition**: If `true`, the loop continues. If `false`, it stops.
3. **Increment/Decrement**: Updates the loop variable after each iteration.

---

## **Example: Print Numbers from 1 to 5**

```go filename="main.go" copy
package main

import "fmt"

func main() {
    for i := 1; i <= 5; i++ {
        fmt.Println(i)
    }
}
```

### **Explanation**

1. `i := 1` initializes `i` to `1`.
2. `i <= 5` checks if `i` is **less than or equal to 5**.
3. `i++` increments `i` by **1** after each loop iteration.
4. The loop **stops when `i` becomes 6**.

**Output:**

```
1
2
3
4
5
```

---

## **Using `for` as a `while` Loop**

Go doesnâ€™t have a `while` loop, but `for` can be used like one.

```go filename="main.go" copy
package main

import "fmt"

func main() {
    num := 1

    for num <= 5 {
        fmt.Println(num)
        num++ // Increment inside the loop
    }
}
```

### **Explanation**

1. `num := 1` initializes `num`.
2. `for num <= 5` is a condition-based loop (like `while`).
3. Inside the loop, `num++` increments `num`.
4. The loop stops when `num > 5`.

---

## **Infinite `for` Loop (Runs Forever)**

```go filename="main.go" copy
package main

import "fmt"

func main() {
    for {
        fmt.Println("This will run forever!")
    }
}
```

### **Explanation**

- No condition means the loop **never stops**.
- Use `break` inside the loop to exit manually.

---

## **Using `break` to Stop a Loop**

```go filename="main.go" copy
package main

import "fmt"

func main() {
    for i := 1; i <= 10; i++ {
        if i == 6 {
            break // Stop the loop when i is 6
        }
        fmt.Println(i)
    }
}
```

**Output:**

```
1
2
3
4
5
```

---

## **Using `continue` to Skip an Iteration**

```go filename="main.go" copy
package main

import "fmt"

func main() {
    for i := 1; i <= 5; i++ {
        if i == 3 {
            continue // Skip when i is 3
        }
        fmt.Println(i)
    }
}
```

**Output:**

```
1
2
4
5
```

**Explanation:**

- When `i == 3`, `continue` **skips** that iteration.
- The loop continues with `i = 4`.

---

## **Conclusion**

- Go only has the **`for`** loop (no `while` or `do-while`).
- It can be used in **three ways**:
  - **Standard `for` loop** (`for i := start; condition; update {}`)
  - **While-like `for` loop** (`for condition {}`)
  - **Infinite loop** (`for {}`)
- **Use `break`** to exit early.
- **Use `continue`** to skip an iteration.
