# 🍃 Overview of MongoDB

## What is MongoDB?

MongoDB is a **NoSQL database** designed for high performance, scalability, and flexibility. It stores data in **JSON-like documents**, making it easy to work with dynamic schemas.

## 🌟 Why Use MongoDB?

✅ **Schema Flexibility** - Stores data in JSON format.
✅ **Scalability** - Supports horizontal scaling.
✅ **High Performance** - Optimized for large-scale applications.
✅ **Rich Query Language** - Supports aggregation, indexing, and complex queries.
✅ **Cloud Integration** - Works seamlessly with cloud services.

## 🏗️ Core Features

- **Document-Based Storage**: Uses BSON (Binary JSON) format.
- **Indexes**: Enhances query performance.
- **Aggregation Framework**: Powerful data processing.
- **Replication & Sharding**: Ensures high availability and scalability.

## 📦 How to Get Started?

1. **Install MongoDB:** [Download Here](https://www.mongodb.com/try/download/community)
2. **Start the MongoDB server:**
   ```sh
   mongod
   ```
3. **Connect using Mongo Shell:**
   ```sh
   mongo
   ```
