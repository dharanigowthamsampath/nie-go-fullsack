## **üîµ 2. Using MongoDB Shell (CLI)**
### **What is MongoDB Shell (mongosh)?**  
MongoDB Shell (mongosh) is a **command-line interface** for interacting with your database using JavaScript commands.

### **Step 1: Install MongoDB Shell**
- If you installed MongoDB, `mongosh` is already available.
- To install it separately, download from:  
  üîó [https://www.mongodb.com/try/download/shell](https://www.mongodb.com/try/download/shell)

### **Step 2: Start MongoDB Shell**
1Ô∏è‚É£ Open **Terminal / Command Prompt**  
2Ô∏è‚É£ Run the command:  
   ```sh
   mongosh
   ```
   ‚úÖ This connects to the default local MongoDB server.

### **Step 3: Basic Commands in MongoDB Shell**
| **Command** | **Description** |
|------------|---------------|
| `show dbs` | Lists all databases |
| `use mydb` | Switches to **mydb** database |
| `db.createCollection("users")` | Creates a collection named **users** |
| `db.users.insertOne({name: "Alice", age: 25})` | Inserts a document into **users** |
| `db.users.find()` | Displays all documents in **users** |
| `db.users.updateOne({name: "Alice"}, {$set: {age: 26}})` | Updates a document |
| `db.users.deleteOne({name: "Alice"})` | Deletes a document |
| `show collections` | Lists all collections in the database |
